# Mosque Digital Clock

Sistem Jam Digital Masjid modern berbasis Next.js dengan fitur:
- Jadwal Sholat & Imsakiyah otomatis via Kemenag API (atau Koordinat).
- Hitung mundur Iqamah.
- Tampilan Murottal/Audio sebelum Azan.
- Slider pengumuman & Galeri Media.
- Dashboard Admin yang aman.

## Struktur Project
- `apps/web-client`: Aplikasi display utama.
- `apps/web-admin`: Dashboard manajemen.
- `packages/shared-types`: Tipe data bersama.

## Cara Menjalankan
1. `npm install`
2. `npm run dev`
